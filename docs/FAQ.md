# Frequently Asked Questions

**Q: Is this really free?**
Yes. MIT licensed. Self-hosted on your own Supabase account. Zero monthly fees.

**Q: Do I need coding knowledge?**
Basic terminal knowledge helps for setup. Once running, everything is managed through the dashboard UI.

**Q: Can I use Stripe instead of Razorpay?**
Stripe support is on the roadmap. Currently Razorpay is supported out of the box.

**Q: How many contacts can it handle?**
Supabase free tier supports up to 500MB database. That's roughly 50,000+ contacts.

**Q: Can I use Gmail for sending?**
Yes, any SMTP provider works including Gmail, but dedicated email services like Resend are recommended for deliverability.

**Q: Is WhatsApp supported?**
WhatsApp Cloud API integration is on the roadmap.

**Q: Can multiple coaches use one installation?**
Multi-tenant support is planned. Currently it's single-tenant.
